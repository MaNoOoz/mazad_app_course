Getting Started


-- Backend Required  https://github.com/MaNoOoz/backend

Clone the repository 

    git clone https://github.com/MaNoOoz/mazad_app_course.git
    Navigate to cloned folder 
    Get packages using flutter pub get
    Run flutter run
